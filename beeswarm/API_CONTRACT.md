# Classification Prediction API Contract

This document defines the API contract for the classification prediction system. Use this when implementing the real model API.

## Base Structure

All predictions follow this structure:

```typescript
interface ClassificationAlert {
  id: string;                 // Unique alert ID (generated by API)
  hiveId: string;            // Hive identifier
  severity: "Critical" | "Warning" | "Info";
  title: string;             // Human-readable alert title
  message: string;           // Detailed alert message
  timestamp: string;         // ISO 8601 timestamp
  classification: string;    // Machine classification type
}
```

## Endpoints

### 1. Get Prediction for Single Hive

**Request:**
```
GET /api/v1/hives/{hiveId}/prediction
Authorization: Bearer {authToken}
```

**Response (200 OK):**
```json
{
  "id": "alert_1234567890_abc123",
  "hiveId": "hive-1",
  "severity": "Warning",
  "title": "Pre-swarm Activity Detected",
  "message": "Model detected pre-swarm behavior patterns. Monitor closely for swarm preparations.",
  "timestamp": "2026-04-20T10:30:00Z",
  "classification": "pre-swarm"
}
```

**Error Response (401 Unauthorized):**
```json
{
  "error": "Invalid or expired authentication token"
}
```

### 2. Get Predictions for Multiple Hives

**Request:**
```
GET /api/v1/hives/predictions?hiveIds=hive-1,hive-2,hive-3
Authorization: Bearer {authToken}
```

**Response (200 OK):**
```json
[
  {
    "id": "alert_1234567890_abc123",
    "hiveId": "hive-1",
    "severity": "Critical",
    "title": "Swarm Alert",
    "message": "Swarm activity detected! Immediate action may be required.",
    "timestamp": "2026-04-20T10:35:00Z",
    "classification": "swarm"
  },
  {
    "id": "alert_1234567891_abc124",
    "hiveId": "hive-2",
    "severity": "Info",
    "title": "Hive Status Normal",
    "message": "Hive is operating normally with no issues detected.",
    "timestamp": "2026-04-20T10:35:00Z",
    "classification": "healthy"
  }
]
```

## Classification Types

The API should return one of these classification values:

| Classification | Severity | Meaning |
|---|---|---|
| `healthy` | Info | Hive operating normally |
| `pre-swarm` | Warning | Pre-swarm behavior detected |
| `swarm` | Critical | Swarm event detected |
| `disease` | Critical | Disease symptoms found |
| `mite_infestation` | Warning | High mite levels detected |
| `queen_loss` | Critical | Queen loss suspected |

## Severity Levels

- **Critical** - Requires immediate attention (red alert, 5s display)
- **Warning** - Should be addressed soon (orange alert, 5s display)
- **Info** - Informational only (green alert, 4s display)

## Integration Steps

1. **Update mockPredictionApi.ts:**

```typescript
export async function getPrediction(
  hiveId: string,
  authToken: string
): Promise<ClassificationAlert> {
  const response = await fetch(
    `https://your-api.com/api/v1/hives/${hiveId}/prediction`,
    {
      method: "GET",
      headers: {
        Authorization: `Bearer ${authToken}`,
        "Content-Type": "application/json",
      },
    }
  );

  if (!response.ok) {
    throw new Error(`API error: ${response.status} ${response.statusText}`);
  }

  const data = await response.json();
  
  // Validate response structure
  if (!data.id || !data.hiveId || !data.severity) {
    throw new Error("Invalid API response format");
  }

  return data as ClassificationAlert;
}
```

2. **Update configuration:**

```typescript
const MOCK_API_CONFIG = {
  enabled: false,  // Disable mock mode
  baseUrl: "https://your-api-endpoint.com",
  delayMs: 0,
};
```

3. **Pass authentication token to hooks:**

```typescript
// Store auth token after login
const [authToken, setAuthToken] = useState<string | null>(null);

// Pass to fetch functions
const prediction = await getPrediction(hiveId, authToken!);
```

## Response Time Requirements

- Single hive prediction: < 500ms
- Multiple hives (10): < 1000ms
- Should be cacheable for 10-30 seconds

## Rate Limiting

Recommended:
- 100 requests per minute per user
- Return 429 Too Many Requests if exceeded
